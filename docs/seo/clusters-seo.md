# Clusters SEO — THIASA Reformas (Madrid)

> Definición de los 14 clusters temáticos que organizan todo el contenido del blog.
> Cada cluster indica: página pilar, artículos del calendario, keywords objetivo y estructura de enlaces.

Última actualización: 2026-07-10

---

## Cómo leer cada cluster

- **Pilar:** página del sitio que actúa como hub del cluster (recibe enlaces de los spokes).
- **Spokes:** artículos del blog (`calendario-publicaciones.csv`) que apoyan al pilar.
- **Keywords objetivo:** ver `keywords.csv` para el detalle y prioridad.
- **Enlaces:** cada spoke enlaza al pilar + 1–2 spokes relacionados.
